Vinyl Vault

Music · Vinyl Records

Vinyl Vault is a personal, single-file web app for browsing and managing a vinyl record collection synced from Discogs. It's essentially a personal Discogs client focused on browsing and collection-value tracking, built as a single portable HTML file with no backend — all state either comes live from Discogs or lives in the browser's local storage.
